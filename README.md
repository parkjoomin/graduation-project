## 실행 방법

1. 프로젝트 레포지토리를 클론합니다.

```sh
$ git clone [https://github.com/parkjoomin/graduation-project.git]
```

2. 프로젝트 실행에 필요한 패키지를 설치합니다.

```sh
$ cd front
$ npm i

$ cd back
$ npm i
```

3. server을 실행합니다.
```sh
$ npm start
```
